## Installation

### Clone repo and go to directory

```bash
git clone https://github.com/syfqpie/sb-toolkit.git
cd sb-toolkit
```

### Create virtualenv

```bash
virtualenv env
```

### Activate virtualenv (Windows)

```bash
env/Scripts/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```
